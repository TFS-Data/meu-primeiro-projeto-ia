## 📝 Sobre o Projeto
Este projeto é um protótipo de uma landing page para fonoaudiologia, desenvolvido durante a **primeira aula do curso Desenvolvimento de Aplicações com IA Generativa utilizando Google Antigravity no SENAI**. O objetivo foi aplicar conceitos de design responsivo e utilizar IA Generativa para acelerar a estruturação de projetos web profissionais.

## 🛠️ Tecnologias e Ferramentas
- **IA Generativa:** Estruturação de código e criação de conteúdo estratégico.
- **HTML5 & CSS3:** Estruturação moderna com foco em UX (User Experience).
- **Google Fonts & Phosphor Icons:** Tipografia e iconografia refinadas.
- **GitHub Pages:** Hospedagem e deploy do projeto.

## 📈 Aprendizados e Evolução
Neste projeto, pude praticar:
- O uso de **Prompts Avançados** dentro da metodologia de ensino do SENAI.
- Gestão de repositórios e controle de versão no **GitHub**.
- Resolução de bugs de deploy e ajuste de caminhos de arquivos.
- Design focado em conversão e acessibilidade.
